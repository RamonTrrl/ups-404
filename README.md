# 🔍 ups-404 – Página personalizada de error 404

## 📸 Vista previa

![Vista previa del proyecto](https://raw.githubusercontent.com/RamonTrrl/ups-404/main/ups_alt_404-removebg-preview.png)

## 📝 Descripción

Página web personalizada para el error HTTP 404 (página no encontrada), diseñada para ser alojada en mi [blog personal](https://cronicasdeilusos.blogspot.com) y en la [revista que edito junto a "los hnos TV"](https://jifinetcom.wordpress.com/acerca-de/). La idea es ofrecerte una experiencia amigable y creativa cuando tratas de visitarnos, pero el browser dominante actual de búsquedas al no tenerno buena estima, pues te lleva a una ruta inexistente.

Este proyecto forma parte de mi blog:  
👉 [cronicasdeilusos.blogspot.com](https://cronicasdeilusos.blogspot.com)

## ✨ Características

- ✅ Diseño responsivo
- ✅ Mensaje claro y amigable
- ✅ Enlace de retorno al inicio del blog
- ✅ Estilo visual acorde al blog principal

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (si aplica)

## 📁 Estructura del proyecto

*(Ajusta los nombres de archivo según los que tengas en el repositorio)*

- `index.html` – Página principal del error 404
- `styles.css` – Estilos visuales
- `ups_alt_404-removebg-preview.png` – Imagen de vista previa

## 🚀 Cómo usarlo

1. Clona este repositorio o descarga los archivos.
2. Sube el contenido a tu servidor o a GitHub Pages.
3. Configura tu servidor (o plataforma de hosting) para que redirija las páginas no encontradas a este archivo `index.html`.
