# 🔍 Ups-404 – Página de error 404 personalizada

## 📸 Vista previa

![Ups 404](https://raw.githubusercontent.com/RamonTrrl/ups-404/main/ups_alt_404-removebg-preview.png)

## 📝 Descripción

Página web personalizada para el error HTTP 404 (página no encontrada), diseñada para ser alojada en mi blog personal. Ofrece una experiencia amigable y creativa cuando un visitante llega a una ruta inexistente.

## ✨ Características

- ✅ Diseño responsivo
- ✅ Mensaje claro y amigable
- ✅ Enlace de retorno al inicio del blog
- ✅ Estilo visual acorde al blog principal

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- (Opcional: JavaScript si aplica)

## 📁 Estructura del proyecto

- [`index.html`](https://github.com/RamonTrrl/ups-404/blob/main/index.html) – Página principal del 404
- [`styles.css`](https://github.com/RamonTrrl/ups-404/blob/main/styles.css) – Estilos visuales
- [`ups_alt_404-removebg-preview.png`](https://github.com/RamonTrrl/ups-404/blob/main/ups_alt_404-removebg-preview.png) – Imagen de fondo o ilustración

## 🚀 Cómo usarlo

1. Clona el repositorio.
2. Abre `index.html` en tu navegador.
3. Personaliza los mensajes o enlaces según tu blog.
